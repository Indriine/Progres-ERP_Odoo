# "Progress Odoo_Toko Roti" #
Pada project ini saya melakukan implementasi sistem ERP Odoo untuk operasional toko roti.

# 1. Modul yang digunakan
- Sales
- Invoicing
- Purchase
- Inventory
- Manufacturing

# 2. Alur Bisnis Toko Roti
- Vendor -> Purchase Order -> Bahan diterima -> Manufacturing Order -> Produk Jadi -> Sales Order -> Kirim ke pelanggan.
